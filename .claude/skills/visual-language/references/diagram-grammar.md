# Diagram Grammar

Use a limited symbol set so the audience learns the visual language quickly.

## Suggested Roles

| Meaning | Recommended visual |
|---------|--------------------|
| Actor / system | rounded container or labeled icon |
| Data / artifact | document-like card or neutral rectangle |
| Process / transform | directional node with outgoing connector |
| Evidence / metric | highlighted numeric block or chart callout |
| Warning / limitation | warm accent badge or outlined callout |
| Transition | thin bridge line, step label, or section card |

## Connector Rules

- solid arrows for causal or process flow
- thin lines for association only
- do not mix three arrow styles on one slide unless meaning truly differs
- reserve accent color for the connector that carries the main story path

## Repetition Rules

- the same concept should keep the same shape family and color role across slides
- if a concept changes state, vary fill or border before changing the base shape
- keep the number of symbol families small enough to memorize in one viewing
