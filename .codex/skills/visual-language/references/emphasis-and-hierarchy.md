# Emphasis And Hierarchy

Build hierarchy with a small set of repeatable cues.

## Hierarchy Tools

Use these levers intentionally:
- size
- contrast
- position
- whitespace
- reveal order
- repetition

## Rules For Focus

- each slide should have one dominant focal point
- supporting elements should remain visibly subordinate
- if two things compete equally for attention, the slide likely has two messages
- use whitespace to separate groups before adding more borders or decoration

## Emphasis Rules

- strongest emphasis goes to the current claim, result, or next reasoning step
- combine at most a few emphasis signals at once; do not stack huge size, saturated color, bold border, and motion on the same minor element
- use repetition for structure, then break repetition intentionally for the one thing that matters most
- active titles and explicit takeaway text should reinforce the same hierarchy as the visuals

## Review Questions

- where does the eye land first?
- does that match the spoken point?
- can the slide be understood in layers rather than all at once?
- does the visual order match the reading order?
