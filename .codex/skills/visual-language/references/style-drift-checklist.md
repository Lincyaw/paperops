# Style Drift Checklist

Use this checklist in review to decide whether the deck has lost visual coherence.

## Palette Drift

- the same color means different things on different slides
- highlight colors appear on low-priority elements without a semantic reason
- too many saturated colors compete at once

## Symbol Drift

- the same concept changes shape family or icon metaphor without explanation
- equivalent states are styled differently slide to slide
- connector styles change without meaning changing

## Layout Drift

- equivalent slide types use unrelated skeletons without a deliberate reason
- margins, spacing, or card structures vary randomly across repeated slide families
- comparison slides do not align analogous elements consistently

## Emphasis Drift

- different slides use different emphasis logic for the same semantic role
- secondary content is louder than the main claim
- every slide tries to emphasize too many things at once

## Image Drift

- screenshots, plots, icons, and photos feel like they come from different visual worlds
- decorative imagery appears only on some slides and changes the deck tone
- annotations vary wildly in color, line weight, or labeling style

## Decision Rule

If the deck feels inconsistent, name the drift precisely before changing layout code.
